# Stockifi Design System

Component demos for the Stockifi design system. Each component is a standalone Vite + React + TypeScript project.

## Running a demo locally

```bash
cd component-demos/dialog
npm install
npm run dev
```

## Structure

```
component-demos/
  dialog/        — Dialog component (input · alert · confirmation)
  overlay-drawer/  — coming soon
```

## Stack

- Vite + React 18 + TypeScript
- Design tokens from Stockifi Component Library (Figma)
